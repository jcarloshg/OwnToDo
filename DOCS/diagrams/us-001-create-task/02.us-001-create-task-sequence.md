# Sequence Diagram - US-001: Create a New Task

## Happy Path Diagram

```mermaid
sequenceDiagram
    actor User
    participant Client as "React Frontend"
    participant API as "Express API"
    participant DB as "PostgreSQL"

    User->>+Client: Click "Create Task" button
    Client-->>User: Display task form

    User->>+Client: Enter task details
    Note over Client: Title, effort points<br/>description (optional)<br/>due date (optional)

    Client->>+API: POST /api/tasks
    Note over API: Validate input:<br/>- Title required (1-200 chars)<br/>- Effort points (1-100)<br/>- Description max 2000 chars

    API->>+DB: INSERT INTO tasks
    DB-->>-API: Task created with ID
    deactivate DB

    API-->>-Client: 201 Created + task data
    deactivate API

    Client-->>User: Display success toast
    Note over Client: "Task created successfully"<br/>Update task list
    deactivate Client
```

```mermaid
sequenceDiagram

    Client->>+API: POST /api/tasks {title, effort: 5}

    DB-->>-API: RETURNING *
    deactivate DB
    API-->>-Client: 201 {id, title, effort, status: "pending"}
    end

    rect rgb(255, 255, 191)
    note over User,DB: UI Update Phase (~100ms)
    Client->>User: Show "Task created successfully"
    Client->>User: Add task to list
    deactivate Client
    end

    Note over User,DB: Total perceived time: ~5-10s (user input dominant)
```

## Complete Flow with ALT Fragment

```mermaid
sequenceDiagram
    actor User
    participant Client as "React Frontend"
    participant API as "Express API"
    participant DB as "PostgreSQL"

    User->>+Client: Click "Create Task"
    Client-->>User: Display empty task form

    User->>Client: Fill form and click "Save"
    Client->>+API: POST /api/tasks

    API->>API: Validate input

    alt Validation Success
        API->>+DB: INSERT INTO tasks (title, effort, description, due_date, status, user_id)
        DB-->>-API: Task object with ID
        API-->>-Client: 201 Created
        Client->>User: Show success toast<br/>Add task to dashboard
    else Validation Failure
        API-->>-Client: 400 Bad Request
        Client->>User: Show error message<br/>"Title is required"
    else Server Error
        API-->>-Client: 500 Internal Server Error
        Client->>User: Show error toast<br/>"Something went wrong"
    end

    deactivate Client
```

## Component Responsibilities

| Component | Responsibility |
|-----------|----------------|
| **React Frontend** | Capture user input, display forms/messages, handle UI state |
| **Express API** | Validate input, orchestrate database operations, return responses |
| **PostgreSQL** | Persist task data with timestamps and relationships |

## Message Summary

| Step | Message | Type | From → To |
|------|---------|------|-----------|
| 1 | Click "Create Task" | Request | User → Client |
| 2 | Form displayed | Response | Client → User |
| 3 | Enter details | Async | User → Client |
| 4 | POST /api/tasks | Request | Client → API |
| 5 | INSERT query | Request | API → DB |
| 6 | Task created | Response | DB → API |
| 7 | 201 Created | Response | API → Client |
| 8 | Success toast | Display | Client → User |
