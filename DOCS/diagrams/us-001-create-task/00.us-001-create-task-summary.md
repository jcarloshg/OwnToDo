# US-001: Create a New Task - Summary

## 📌 Overview

| Field          | Value                                                                                                            |
| -------------- | ---------------------------------------------------------------------------------------------------------------- |
| **User Story** | As a user, I want to create a new task with effort points, so that I can track the work needed for my activities |
| **Epic**       | Task Management                                                                                                  |
| **Priority**   | P0 (MVP)                                                                                                         |
| **Estimable**  | ~3 days (frontend + API)                                                                                         |

---

## 📝 Acceptance Criteria (Gherkin)

### Scenario 1: Create task with all fields

- **Given** I am logged into the application and on the dashboard
- **When** I click the "Create Task" button
  - And I enter "Complete project report" as the task title
  - And I enter "Write the final Q1 report for the marketing team" as the description
  - And I enter "5" as the effort points
  - And I select "2026-04-25" as the due date
  - And I click the "Save" button
- **Then** I should see a success message "Task created successfully"
  - And the task "Complete project report" should appear in my task list
  - And the task should have 5 effort points

### Scenario 2: Create task with minimal fields

- **Given** I am logged into the application and on the dashboard
- **When** I click the "Create Task" button
  - And I enter "Buy groceries" as the task title
  - And I enter "2" as the effort points
  - And I click the "Save" button
- **Then** I should see a success message "Task created successfully"
  - And the task "Buy groceries" should appear in my task list

### Scenario 3: Validation failure

- **Given** I am logged into the application and on the dashboard
- **When** I click the "Create Task" button
  - And I click the "Save" button
- **Then** I should see an error message "Title is required"
  - And the form should remain open

## 🎯 Use Cases Breakdown

| Use Case                    | Type        | Description                                                               |
| --------------------------- | ----------- | ------------------------------------------------------------------------- |
| **UC1: Create New Task**    | Primary     | Main actor interaction - user clicks create and fills form                |
| **UC2: Enter Task Details** | <<include>> | Capture title, effort points, description (optional), due date (optional) |
| **UC3: Validate Input**     | <<include>> | Check title (1-200 chars required), effort (1-100 integer)                |
| **UC4: Save Task**          | <<include>> | Persist task to PostgreSQL with status "pending" and timestamps           |
| **UC5: Show Success**       | <<include>> | Display toast "Task created successfully", update task list               |
| **UC6: Show Error**         | <<extend>>  | Display validation error message, form remains open                       |
